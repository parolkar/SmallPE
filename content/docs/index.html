AI-First Investment Team Architecture

Orchestrated Intelligence

The SmallPE Agent Swarm consists of five specialized AI agents, each with distinct roles and capabilities, working together under the coordination of a Managing Partner agent. This architecture enables parallel processing, specialized expertise, and seamless collaboration across the entire investment lifecycle.

Each agent operates with advanced AI models (Claude Opus/Sonnet), has access to comprehensive toolsets, and maintains dedicated workspaces for their specialized functions.

Key Features:

  • • Parallel processing capabilities
  • • Specialized domain expertise
  • • Automated workflow orchestration
  • • Ruby-based tool development

The Team of AI Agents

Managing Partner

Strategic Orchestrator & Decision Maker

Core Responsibilities:

  • • Coordinate entire deal pipeline across all agents
  • • Make strategic investment decisions
  • • Prioritize deal opportunities and resource allocation
  • • Ensure alignment between thesis and execution
  • • Oversee AI-native competitive advantage

Technical Capabilities:

  • • Model: Claude Opus
  • • Connections: All agents (Oracle, Athena, Helios, Logos, Hermes)
  • • Special Tools: Pandoc MCP for document conversion
  • • Full toolset: Read, Edit, Write, Bash, WebFetch, WebSearch

Focus: Building SmallPE as an AI-native firm with intelligent automation at its core, creating a flywheel effect where every deal makes the firm smarter.
